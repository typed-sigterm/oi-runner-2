# 更新记录

## 1.1.0 (2024-11-17)

### 🚀 新功能

- 更新了输入框样式。我们从 [CodeMirror](https://codemirror.net/) 迁移到了 [Monaco Editor](https://microsoft.github.io/monaco-editor/)，获得了更好的输入体验，并且终于修复了 Ctrl+V 粘贴不生效的问题
- 在运行面板中显示当前运行的文件，以及文件是否已保存
- 改进了工具栏按钮 hover 时的的背景形状

### 🩹 Bug 修复

- 修复加载动画和遮罩的位置
- 输入框滚动时不影响上方的标题
- 遮罩显示时阻止滚动

### 📖 文档改进

- 优化版本标签外观
